# Garatujas

Garatujas são os primeiros rabiscos, traços desordenados ou desenhos rudimentares feitos por crianças, marcando o início da expressão gráfica e do desenvolvimento da coordenação motora entre 0 a 6 anos. Representam o mundo ao redor, estimulando criatividade e comunicação pré-verbal.

# HTML para o eu do Futuro, ou do presente também

HTML (HyperText Markup Language) é a linguagem de marcação padrão para estruturar páginas web, agindo como o "esqueleto" de um site ao definir elementos como títulos, parágrafos, imagens e links. Utiliza tags para organizar o conteúdo, sendo essencial para navegadores exibirem informações corretamente.

# CSS para o eu do Futuro, ou do present também

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é a linguagem de estilo utilizada para definir a apresentação visual de documentos HTML ou XML. Ela controla o layout, cores, fontes, espaçamentos e responsividade, separando a estrutura do conteúdo (HTML) do design.
